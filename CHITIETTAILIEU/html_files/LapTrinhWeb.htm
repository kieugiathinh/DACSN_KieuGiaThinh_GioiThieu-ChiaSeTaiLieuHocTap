LẬP TRÌNH WEB

Lập trình web là môn học nghiên cứu cách phát triển các ứng dụng và trang web động, bao gồm cả phía máy khách (frontend) và phía máy chủ (backend). Môn học tập trung vào các công nghệ như HTML, CSS, JavaScript, và các framework như React, Angular, Node.js, cùng với việc xây dựng cơ sở dữ liệu và xử lý yêu cầu từ người dùng. Mục tiêu là tạo ra các ứng dụng web mạnh mẽ, tương tác và dễ sử dụng.

Nội dung tài liệu

Chương 1: Hướng dẫn học và thực hành
  • Phần mềm hỗ trợ lập trình, hướng dẫn học và cách mở mã nguồn ví dụ.
Chương 2: Các khái niệm cơ bản về HTML
  • Giới thiệu về HTML, cấu trúc tài liệu HTML, và các thẻ HTML cơ bản.
Chương 3: Các thẻ HTML hay dùng
  • Chi tiết về các thẻ như <div>,<p>, <ul>, <h1>, <a> và cách sử dụng chúng.
Chương 4: Thẻ Inline và Block
  • Phân biệt giữa thẻ inline và thẻ block trong HTML.
Chương 5: Tổng kết về HTML
  • Tóm tắt các kiến thức đã học về HTML.
Chương 6: Khai báo CSS
  • Cách khai báo CSS và các bộ chọn CSS.
Chương 7: Định dạng nội dung văn bản
  • Cách định dạng văn bản và các thuộc tính CSS liên quan.
Chương 8: Mô hình hộp Box Model
  • Giới thiệu về mô hình hộp và các thuộc tính liên quan như margin, padding, border.
Chương 9: Float và Clear trong CSS
  • Cách sử dụng thuộc tính float và clear để bố trí các phần tử.
Chương 10: Định dạng một số thẻ HTML đặc biệt
  • Cách định dạng các thẻ HTML đặc biệt như <a> và <ul>.
Chương 11: Giới thiệu về HTML5 và CSS3
  • Các tính năng mới trong HTML5 và CSS3.
Chương 12: Thực hành thiết kế giao diện HTML - CSS
  • Hướng dẫn thực hành thiết kế giao diện web bằng HTML và CSS.

Tài liệu Lập trình Web
